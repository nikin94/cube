Задача: Построение интерактивного трехмерного кубика Рубика с помощью библиотеки three.js для запуска через браузер. Кубик Рубика представляет собой фигуру, состоящую из n*n*n слоев, каждый из наружных окрашен в один из 6-ти установленных цветов.
Использование: Повороты граней и слоев с помощью мыши и консоли - при зажатии LMB над одним из элементов куба и ведении в одну из сторон должен происходить поворот слоя с данным элементом в этом же направлении. Должна быть возможность перемешивать куб, т.е. случайным образом вращать разные слои до состояния с максимально рандомизированого положения элементов на поверхности куба, а также изменение формы и размеров куба - возможность задания размерности куба начиная с двух.
Требования: Цвета граней должны соответствовать стандартной, установленной WCA(World Cube Association) цветовой схеме. Так же ведение лога движений граней, где движение каждой грани по часовой стрелке на 90 градусов должно обозначаться заглавной латинской буквой R,L,U,D,F,B(right, left, up, down, front, back соответственно). При движении против часовой стрелки к букве приписывается апостроф. при повороте на 180 градусов перед буквой ставится "2"(прим. RU2FRF'B'U).
Критерии тестирования результата: Корректные повороты граней с верными записями в логах, возможность вращения и масштабирования камеры с помощью мыши - управление, аналогичное повороту граней, но с зажатой клавишей CTRL - при вращении камеры грани куба не могут вращаться, а так же возможность работы одновременно 2х кубиков на одной странице.
