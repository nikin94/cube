Задача: Построение интерактивного трехмерного кубика Рубика с помощью библиотеки three.js для запуска через браузер.
Использование: Повороты граней с помощью мыши и консоли, перемешивание, а также изменение формы и размеров куба.
Требования: Цвета граней должны соответствовать стандартной, установленной WCA(World Cube Association) цветовой схеме. Так же ведение лога движений граней, где движение каждой грани по часовой стрелке на 90 градусов должно обозначаться заглавной латинской буквой R,L,U,D,F,B(right, left, up, down, front, back соответственно). При движении против часовой стрелки к букве приписывается апостроф. при повороте на 180 градусов перед буквой ставится "2"(прим. RU2FRF'B'U).
Критерии тестирования результата: Корректные повороты граней с верными записями в логах, возможность вращения и масштабирования камеры, а так же возможность работы одновременно 2х кубиков на одной странице.
